# **App Name**: CompanyFlow

## Core Features:

- Employee Management: CRUD operations for employee profiles: name, project, enrollment date, roles, etc.
- Task Assignment: Assign tasks to employees and track their progress on specific projects.
- Project Overview: Dashboard providing a high-level overview of all active projects.
- Attendance Tracking: Record and manage employee attendance and leave requests.
- Meeting Summarization Tool: AI-powered tool that generates summaries of recorded meeting audio.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey stability and trust.
- Background color: Light grey (#F0F2F5) for a clean and modern look.
- Accent color: Soft purple (#9575CD) for highlighting interactive elements.
- Headline font: 'Poppins', a geometric sans-serif for headlines and short amounts of text.
- Body font: 'PT Sans', a humanist sans-serif for the main body text.
- Use rich, structured icons from a library like 'Lucide' to provide visual cues.
- Implement a consistent and structured layout with clear visual hierarchy using Tailwind CSS and Shadcn components.
- Subtle animations for UI transitions and interactions to enhance user experience.
